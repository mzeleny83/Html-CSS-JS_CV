Currently, I want to develop a mobile application Carpooling oriented to everyday commuting to work within the city, however it would be used even for non work voyages. The goal is that citizens from the neighbourhood connect together and agree to commute together on a regular basis either for work or for non-work ways. Into the application, it would be entered start and end destination and departure time and the app would then match supply and demand for rides in a set distance circle. The result would be a reduction of the traffic load in cities.

For this purpose, I am looking for IT developers with the following focus:

  • Kotlin developer
  • Swift or Objective-C developer
  • JavaScript developer
  • Android Studio, Xcode, React Native, Flutter or Ionic developer
  • RESTful a GraphQL API developer
  • Node.js, Python (Django/Flask), Ruby on Rails or Java (Spring) developer
  • SQL (e.g.. SQLite for mobile database) and NoSQL database (e.g. Firebase, Realm) developer
  • Model-View-ViewModel (MVVM), Model-View-Presenter (MVP) developer
  • Interface Builder and Storyboards developer
  • Flutter developer
